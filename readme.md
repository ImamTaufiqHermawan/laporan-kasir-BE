# RESTful API dengan express sequelize ejs

## Install dependency

```bash
# Pengguna NPM
npm install
```

## config env variable
jangan lupa config env

## run sequelize

```
npx sequelize db:create (kalau gk ada db nya)
npx sequelize db:migrate
npx sequelize db:seed:all
```

## Jalankan server

```
npm run dev
```
